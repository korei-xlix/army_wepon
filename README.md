# 軍事兵器 辞書

# このドキュメントについて <a name="aHowto"></a>
このドキュメントは世界の軍事兵器のうち、わたしが気になったものについてピックアップデータをまとめていきます。  
主にアズールレーン、陸上自衛隊の兵器、世界情勢でピックアップされた兵器について記載します。  
なおデータは主にWikipediaを参照しています。
